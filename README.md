# Simple Fly Mod

A simple fly mod for Minecraft 1.17.1 using the Fabric API.

## Features
- Basic flight functionality.
- Compatible with Minecraft 1.17.1.

## Installation
1. Install [Fabric Loader](https://fabricmc.net/use/).
2. Download the latest release from the [Releases](https://github.com/AnNgTv/FlyMod-1.17.1-Fabric/releases) page.
3. Place the JAR file in your `.minecraft/mods` folder.

## Requirements
- Minecraft 1.17.1
- Fabric Loader >= 0.11.3
- Java 16
