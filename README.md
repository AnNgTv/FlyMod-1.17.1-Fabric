# Simple Fly Mod (Minecraft 1.17.1)

A lightweight and efficient flight mod for Minecraft 1.17.1, built using the Fabric API. This mod allows players to toggle flight mode easily in survival or creative.

## 🚀 Features
- **Toggle Flight:** Press `R` (default) to turn flight on/off.
- **In-game Notifications:** Displays a message on the action bar when toggled.
- **Fabric Powered:** Built for stability and performance on the Fabric Loader.

## 🛠️ Installation
1. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft 1.17.1.
2. Download the latest `FlyMod-1.0.0.jar` from the [Releases](https://github.com/AnNgTv/FlyMod-1.17.1-Fabric/releases) page.
3. Place the JAR file in your `.minecraft/mods` folder.
4. (Optional) Install [Fabric API](https://www.curseforge.com/minecraft/mc-mods/fabric-api/files/3371442) for full compatibility.

## ⌨️ Controls
- **Toggle Flight:** `R` (Can be rebound in Options -> Controls).

## 📋 Requirements
- **Minecraft:** 1.17.1
- **Fabric Loader:** >= 0.11.3
- **Java:** 16 or higher
